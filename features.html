üö® Troubleshooting

Common Validation Issues

Duplicate Detection False Positives

Issue: System reports duplicates that seem different

Solution: Check for extra spaces, different cases, or hidden characters

Phone Number Validation Failures

Issue: Valid phone numbers are rejected

Solution: Use international format (+country-area-number) or standard US format

Batch Processing Stops

Issue: Batch processing halts unexpectedly

Solution: Check browser console for errors, reduce batch size, or process files individually

Performance Tips

  • Large Files: Process large files individually for better performance
  • Memory Issues: Close other browser tabs during batch processing
  • Network: Ensure stable internet connection for file downloads
  • Browser: Use Chrome or Firefox for best performance
